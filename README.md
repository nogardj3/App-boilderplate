# boilerplate-android

Android Boilerplate Application

## About this Project

### Introduction

- 안드로이드 샘플 앱
- MVVC, Repostiroy 패턴 기반

- JDK 1.8
- Android Studio Arctic Fox
- Android SDK
- Android 12 (API 31)
- Latest Android SDK Tools and build tools.

- AndroidX Databinding

- AndroidX Navigation
- AndroidX Room
- AndroidX Activity, Fragment
- RxJava3
- Retrofit2
- Glide v4
- Hilt
- Mockito
